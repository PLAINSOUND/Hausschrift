# Hausschrift

_Hausschrift_ – German, “House font”

BETA. Regular: 475 glyphs. Italic: 384 glyphs.

A serif typeface in the Scotch Roman/Century tradition. Started as a redrawing/redesign of LaTeX's “Computer Modern” to better suit use in musical scores. Italic set at 14°. Current OpenType features: ligatures and fractions. Broad Latin-character coverage (with diacritics). Main IPA-range coverage (Regular).
